PRIVACY POLICY

All user information collected by the Sylphy Bot is accessible by any Discord user.

By using the Sylphy Bot, inviting the Bot to your server, using a server that the Bot has joined, or in some other way agreeing to the Sylphy Bot Terms of Service, you are also accepting this privacy policy and agreement. This privacy policy is written to outline the kind of data that the Bot collects and stores, and also how this data is used.

Data we collect:

1. User ID: The identification number attached to all user accounts. User IDs are accessible to anyone with a Discord account, and can be seen after enabling Developer Mode in the Discord settings. We store User IDs in order to identify the artists ("sketchers") of the images contained in the Sylphy Bot search library. We collect User IDs automatically from embedded messages sent by the Karuta Bot.

2. Channel ID: The identification number attached to all guild channels. Channel IDs are accessible to anyone with a Discord account, and can be seen after enabling Developer Mode in the Discord settings. We store Channel IDs to flag specific channels as "display channels". The Bot will automatically display into flagged channels any new entry to the Sylphy Bot's search library as they are entered. We store Channel IDs only at the request of the Guild owner(s) for any guild to which the stored channel belongs, and they are only stored manually.

3. Message Content and Embeds: The Sylphy Bot updates its library by reading "Card" data from Karuta Bot's embedded messages. The data read from these embedded messages includes "Series Names", "Character Names", "Character Aliases", "Card IDs", "Print Numbers", "Edition Numbers", "Card Ink", "Card Inkwells", "Sketcher IDs", and the embedded messages' thumbnail images. This data is accessible to all verified users of the Karuta Bot. This data is only obtained from the Karuta Bot's messages. We analyze Message Content from the Sylphy Bot's users' messages to process text commands, but we do not store any non-bot user message content.


We do not sell or share any collected data with any third parties.

 

DATA DELETION

Users may request at any time for us to remove their User ID or Channel IDs by joining the support server linked in the Sylphy Bot's profile. However, all User IDs stored in the Sylphy Bot are collected from the Karuta Bot's messages, not from the users themselves. Unless a user first requests their data deletion from the Karuta Bot, removing their data from the Sylphy Bot will only temporarily remove their User ID, and the User ID is at risk of being recorded again unless the user requests data deletion from the Karuta Bot first.
